# 3D Packing Algorithm - 三维装箱算法（符合物理模型并且符合体积最大填充利用率

## 项目简介

这是一个基于 C# 开发的三维装箱算法实现，采用多种启发式策略优化空间利用率，支持多容器调度和物理稳定性约束。
### 项目条件
1. 底面积最大的作为底面
2. 并没有最大化利用空间，因为要求叠放不能悬空
3. 模拟的是现实具体装箱条件
4. 牛皮纸填充策略（体积最大利用率）

## 核心特性

### 1. 双算法引擎
- **PlainPackager** - 3D极端点算法
  - 全空间搜索放置点
  - 贪心选择最优位置
  - 适合不规则物品分布
  
- **LaffPackager** - LAFF层级算法  
  - 逐层填充策略
  - 强调底面覆盖
  - 物理稳定性更好

### 2. 多策略优化
- **10种固定策略** - 基于面积/体积/最大维度的排序
- **120种随机策略** - 随机打乱物品顺序尝试
- **并行执行** - 利用多核CPU加速策略搜索

### 3. 物理稳定性约束
- 100%底面支撑检查
- 中心点支撑验证
- 四角支撑验证
- 防止物品悬空

### 4. 多容器智能调度
- 从小到大遍历容器
- 优先装入最多物品
- 同数量选利用率最高
- 自动处理剩余物品
