V4.2.0及之前

本章节介绍4.2.0及其之前版本的IDMVS功能更新动态。

表 1 版本更新说明

版本信息

日期

更新记录

V4.2.0

2024/8/2

  • 功能新增

    • 新增Lua脚本工具。

    • 输入部分新增命令持续触发功能。

    • 输出部分新增指令持续IO输出功能。

    • 通信配置的TCP服务器协议新增心跳相关参数。

    • 历史记录支持导出，并新增过滤无读使能。

    • 快捷工具栏支持调整字体大小。

  • 功能更新

    • 历史记录新增轮询组编号可选项。

    • FTP测试图目录使能功能调整至图像配置模块 。

    • 通信配置的Modbus通信协议支持配置从机地址 。

    • 抓图文件命名规则支持多选并自由配置排列顺序。

    • 设备信息增加本地网卡信息。

  • 优化改进

    • 提升ROI绘制流畅度。

    • 优化参数保存界面。

    • 设备列表去除网卡显示层级，通过分组进行展示。

    • Raw模式支持自动切换 。

    • 优化打码评级子项以及读码报表名称。

    • 优化图像预览窗口的工具栏及按钮显示。

V4.1.0

2024/1/15

  • 优化参数保存界面。

  • 固件升级新增TCP固件升级方式。

  • 支持将本地RAW、BMP格式测试图导入至读码器目录。

  • 按键智能调参支持初次按键实时画面显示。

  • 针对存在初始旋转角度的读码器，支持图像旋转后ROI绘制。

  • 打码评级新增读码质量报告输出功能。

  • 普通过滤新增条码裁剪模式参数。

  • 数据处理新增FTP传输路径格式化配置功能。

  • 码绘制新增焦点锁定、条码分隔符、条码前后缀功能。

  • 新增用户角色权限分配功能。

V4.0.1

2023/9/13

  • 主界面调整预览窗口触发开关位置。

  • 优化时间校准功能，增加手动校时方式。

  • 新增智能调参结果同步参数轮询库功能。

  • 通讯配置中支持蓝牙协议配置和2.4G协议配置。

  • 历史记录新增显示列配置按钮，默认显示触发时间。

  • 通用设置中新增图像自动旋转、报表和读码成功/失败蜂鸣提示音功能。

  • 码绘制中新增条码长度配置功能。

  • IDMVS新增解锁状态图标。

  • 新增FTP服务工具。

  • 正则表达式新增规则，支持正则表达式修改。

V4.0.0

2023/6/17

  • 新增参数轮询、格式化输出、参数保存、记录导出快捷方式。

  • 设备列表新增搜索与排序功能。

  • 属性树参数中英文形式跟随主程序配置。

  • 调整设备重启、相机自动工作参数位置。

  • 删除Raw图运行模式。

  • 码制支持按类别一键勾选/取消勾选，并调整页面布局。

  • 新增鹰眼视图功能。

  • 历史记录新增OCR信息显示列。

  • 新增按照轮询参数组缓存图像数据功能。

  • 优化采集状态显示参数。

  • 新增算法ROI显示/隐藏控制功能。

  • FTP服务支持自动清图功能。

  • 新增定时锁定功能，并优化权限控制功能。

  • 调整日志查看工具为SDK日志工具。

  • 新增日志收集工具。

  • 新增读码训练工具。

V3.2.0

2023/1/8

  • 新增智能调参、自动对焦、自适应调节快捷方式。

  • 新增自动清除存储图片功能，更新不同运行模式下图片保存格式。

  • 客户端新增账户密码登录功能。

  • 新增通信指令手册。

  • 新增开源合规文件导出功能。

  • 新增智能调参功能。

  • UDP/TCP/串口触发下新增字符输入选择功能。

  • UDP/TCP/串口停止触发下新增字符输入选择功能。

  • 调整导航页通信配置与数据处理先后顺序。

  • 前置连续抓图按钮。

  • 新增文件存取工具支持批量导入功能。

  • 新增虚拟读码器录制功能。

V3.1.0

2023/1/8

  • 新增设备私有协议枚举方式。

  • 新增相机列表显示设备过滤功能。

  • 属性树新增数据统计、运行诊断属性。

  • 新增多网卡设备枚举功能。

  • 新增光源可视化展示。

  • 新增ROI区域自动调焦功能。

  • 新增正则表达式过滤规则参数。

  • 统计报表新增读码率曲线。

V3.0.0

2022/3/28

  • 新增设备列表自动枚举功能。

  • 新增历史记录持续保存及自动路径选择功能。

  • 码绘制新增焦点输出功能。

  • 新增FTP服务器功能。

  • 新增ASCII码十六进制查找表。

  • 新增属性树中英文切换功能。

  • 新增设备分组功能。

  • 新增不以特定字符开始/结束过滤规则。

  • IP配置工具中新增设备一键重启功能。

V2.3.1

2021/11/3

  • 新增U口读码器环境配置。

  • 新增识别条码字体大小设置功能。

  • 新增导入导出属性功能。

  • 新增虚拟读码器功能。

  • 新增六路光源功能。

  • 新增自适应调节功能。
