配置调试工具

通过串口调试工具的串口设置模块，配置读码器通信指令参数。

  • 已获取并安装串口调试工具。

    说明

    请联系技术支持获取串口调试工具的安装包。

  • 已完成IDMVS配置，详情见配置客户端。

  • 读码器和PC已通过串口连接线连接。

  1. 选择协议类型为Serial通信协议。
  2. 配置串口通信的串口号、波特率、数据位、校验位及停止位。

    • 串口号：可通过设备管理器 > 端口确认设备串口号。

    • 波特率：与读码器波特率设置一致。

    • 校验位：与读码器波特率设置一致。

    • 数据位：与读码器波特率设置一致。

    • 停止位：与读码器波特率设置一致。

  3. 可选操作： 通过接收设置、发送设置模块可设置数据格式、存储及收发方式。
  4. 单击连接即可完成配置，此时可通过接收区和发送区进行数据收发。

    通过在发送区发送通信指令，可实现对设备进行相应的参数设置，具体指令请参见指令参考表。

  5. 可选操作： 点击断开可断开通信连接。
