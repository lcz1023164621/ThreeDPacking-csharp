快捷键

通过快捷键可对IDMVS的常用功能进行快捷键设置。

完成设置后，可直接通过快捷键实现对应的功能，方便快速操作。

软件常用功能与快捷键的默认关系请见下表。

表 1 常用功能默认快捷键

功能

对应快捷键

连接/断开相机

F1

开始/停止采集

F2

开始/停止预览

F3

单次抓图

Ctrl + P

连续抓图

Ctrl + R

全屏

F4

放大

Ctrl + +

缩小

Ctrl + -

自适应

Ctrl + 1

原比例

Ctrl + 2
说明

具体功能本章节不做展开介绍，请到功能对应章节进行了解。

如需修改默认快捷键，选中功能对应的文本框，同时按下对应的快捷键即可。

说明

  • 修改快捷键时，不能将Delete设为快捷键使用，且不能将不同功能设置为同一个快捷键。

  • IDMVS仅支持将单个键或2个组合键作为快捷键使用。

如需删除快捷键，选中功能对应的文本框，按下Delete即可删除快捷键。此时文本框显示为“None”。

如需恢复默认快捷键，单击恢复默认值即可。
