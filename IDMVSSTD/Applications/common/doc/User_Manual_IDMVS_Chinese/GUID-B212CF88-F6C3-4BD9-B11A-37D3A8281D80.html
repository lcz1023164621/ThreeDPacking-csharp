配置调试工具

通过TCP调试工具的网络设置模块，配置读码器通信指令参数。

  • 已获取并安装TCP调试工具。

    说明

    请联系技术支持获取TCP调试工具的安装包。

  • 已完成IDMVS配置，详情见配置客户端。

  1. 选择协议类型为TCP Client通信协议。
  2. 配置通信的IP地址及端口号。
    • 本地主机地址：保证主机IP地址与读码器IP地址一致；
    • 本地主机端口：保证主机端口与设置的TCP通信端口一致。
  3. 可选操作： 通过接收设置、发送设置模块可设置数据格式、存储及收发方式。
  4. 单击连接即可完成配置，此时可通过接收区和发送区进行数据收发。

    通过在发送区发送通信指令，可实现对设备进行相应的参数设置，具体指令请参见指令参考表。

  5. 可选操作： 点击断开可断开通信连接。
