针对VC demo的运行：

若已执行MvCodeCameraSDK_VX.X.X.X_VC90_Runtime.exe文件，则可直接运行demo的exe;

注意：若未执行Runtime包文件，则需拷贝SDK/C/dll路径下win32或win64文件下的dll和ini配置文件，放置对应Demo的可执行文件exe路径下，即可运行demo的exe。

For the operation of the VC demo:

If executed MvCodeCameraSDK_VX.X.X.X_VC90_Runtime.exe files, can be directly run the demo exe;

Note:
If the Runtime file has not been executed, to run the demo EXE file, you should copy the DLL and INI configuration files from the directory 
SDK/C/dll/win32 or SDK/C/dll/win64 to the directory of demo EXE file.
