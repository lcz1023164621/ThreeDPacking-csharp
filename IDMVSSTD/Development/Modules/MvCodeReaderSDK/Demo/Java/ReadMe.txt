针对Java demo的运行：

运行Demo需安装JDK，Demo基于Java8开发，若在其他Java版本上无法运行请尝试使用Java8运行Demo；

【文件说明】
目录中包含两个Java demo，分别位于\JavaDemo\CallBack及\JavaDemo\GrabImage目录。demo目录下包含了demo的java源码、依赖库、编译脚本及运行脚本。

【操作说明】
1.安装JDK，配置环境变量；
2.点击“编译demo”脚本编辑demo源码；
3.点击“运行demo”脚本运行demo;

For the operation of the Java demo:

Before run the demo,you should install JDK first.The demo is developed based on Java8, if it can't run on the other version of Java, try Java8 again.
