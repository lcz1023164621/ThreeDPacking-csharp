For cross-platform compilation commands:

For example: linux64/GrabImage: grabImage.c (please in ../linux64/GrabImage enter the following command)

compile command: make (if the current user has root permission) / sudo make (if the current user has non-root permission)

run command: ./GrabImage

For example: linux64/Grab_Callback: Grab_ImageCallBack.c (please in ../linux64/Grab_Callback enter the following command)

compile command: make (if the current user has root permission) / sudo make (if the current user has non-root permission)

run command: ./Grab_ImageCallBack


Note: Before compiling and running, copy Include directory and all the xx.so files of the corresponding platform to the path where demo is currently executed

Tips: For the rest of the platforms, compile and perform the sample demo as above
